# Gestione-spese-
Il sistema permette la gestione delle spese personali e del budget mensile attraverso un'interfaccia console in C++ e la persistenza dei dati su database relazionale MySQL
# Progetto Spese Personali
Sistema di gestione spese basato su C++ e SQL.
## Requisiti
- Compilatore C++ (Dev-C++)
- Librerie standard: iostream, string
## Come Compilare ed Eseguire
1. Apri il file `main.cpp` in Dev-C++.
2. Premi **F11** sulla tastiera.
3. Il programma si aprirà e permetterà di inserire i dati tramite menu.
## Funzionamento
Il programma simula l'invio di comandi a un database SQL. Ogni volta che inserisci una spesa o una categoria, il sistema mostra a schermo la "Query SQL" che verrebbe eseguita nel database reale.
